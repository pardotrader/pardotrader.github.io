Pull Back – Análise Gráfica

Aprenda sobre Pull Back, qual o significado na análise técnica, como identificar um pull back no gráfico e a importância desse evento gráfico nas ações.

O que é Pull Back ?

Pull Back = RETORNO = RECUO

O Pull Back é um retorno do ativo após um rompimento, quando ocorre um rompimento e logo em seguida os preços sofrem um recuo até uma região de concentração de ordens (suporte ou resistência).


Aprenda sobre suporte e resistência.

Um exemplo de Pull Back em um rompimento de alta, quando um ativo supera a máxima e depois retorna para preços próximos do rompimento, fazendo um recuo contra tendência de alta para o suporte.

Um exemplo de Pull Back em um rompimento de baixa, quando o ativo supera a mínima e depois retorna para preços próximos do rompimento, fazendo um recuo contra tendência de baixa para a resistência.

O pull back é um evento gráfico  muito importante, pois é uma “segunda chance”, é uma oportunidade de comprar o rompimento de forma atrasada e pelo mesmo preço “ideal”, quase que todos os rompimentos fazem um pull back, então se ocorreu um rompimento e não conseguiu entrar no trade, é possível fazer uma entrada no pull back.
